Guide d'utilisation
📚 Explorer
Browse toutes les leçons par module. Clique sur une leçon pour voir son contenu, modifier ses blocs, changer le quiz ou supprimer une réponse.
✏️ Créer
Créer une leçon from scratch : niveau, module, blocs (hook/contenu/astuce/piège), quiz avec 4 options. Le code JS est généré en temps réel.
🤖 Générer IA
Décris un sujet en langage naturel → l'IA génère la leçon complète au format TACTIC (blocs + quiz 3 questions). Importe en 1 clic.
💡 Idées
12 suggestions basées sur les lacunes du corpus et le programme Digital Learning. Clic → pré-remplit le générateur IA.
📤 Exporter
Exporte le corpus modifié en code JS complet ou en JSON. Colle le code dans ton academy.html pour mettre à jour l'app.
Structure d'une leçon
Hook = phrase choc d'accroche
Contenu = explication principale
Astuce = tip actionnable
Piège = erreur à éviter
Quiz = 4 options, 1 correcte
📥 Importer depuis Tactic Academy
Importe automatiquement les leçons de Tactic Academy (blocs + quiz). Les leçons déjà présentes sont ignorées.
